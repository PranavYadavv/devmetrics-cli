# devmetrics-cli

CLI tool for tracking developer productivity metrics

## Installation

```bash
pip install devmetrics-cli
```

## Quick Start

```python
from devmetrics_cli import Devmetrics
client = Devmetrics()
client.run()
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
